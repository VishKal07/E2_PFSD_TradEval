# TradEval - Trading Evaluation Platform

Real-time trading evaluation platform with ML predictions using real market data from Binance.

## Features

- Real-time market data from Binance exchange
- Machine learning price predictions
- MongoDB database storage
- WebSocket real-time updates
- Clean HTML/CSS/JS frontend
